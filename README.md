A lightweight, high-fidelity HTML5 canvas game inspired by Slither.io. Smooth analog steering (mouse/touch), boost, glowing pellets, and a camera that follows your snake.

### Play

- Open `index.html` in any modern browser.

### Controls

- Mouse/Touch: point to steer
- Space / Double-tap: boost
- Arrow keys / WASD: alternative steering

### Features

- Smooth, framerate-independent turning and motion
- Radial glow, glossy head, soft trail, and starfield
- Large bounded world with grid and edge glow
- Adaptive camera zoom based on length/boost

### Dev Notes

- No build step, single JS file in `src/main.js`
- Assets are procedural; no external dependencies
